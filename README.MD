# GSlerp-CC

This repository is for the paper:

**Generalized and Group Spherical Linear Interpolation for Token-level Context Compression (Neural Networks, 2026)**  
Paper link: https://www.sciencedirect.com/science/article/pii/S0893608026001826

## Overview

GSlerp-CC studies token-level context compression with spherical linear interpolation, and includes two complementary methods:

- **GeSlerp** (Generalized Slerp)
- **GrSlerp** (Group Slerp)

## Repository Structure

- [`GeSlerp`](./GeSlerp): implementation and runnable scripts for GeSlerp.
- [`GrSlerp`](./GrSlerp): codebase for GrSlerp.
